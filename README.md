# PW Gift Cards Balance Viewer Addon for WooCommerce

Adds a **"Gift Cards" section** to the WooCommerce **My Account** page, allowing customers to easily view their PW Gift Card details including balance, usage, and expiration.

## 🚀 Features

- Adds a **Gift Cards tab** in WooCommerce My Account
- Displays gift card details:
  - Gift Card Code
  - Original Amount
  - Remaining Balance
  - Used Amount
  - Issue Date
  - Expiration Date
- Automatically shows gift cards assigned to the logged-in user
- Clean and simple table layout
- Seamless integration with WooCommerce account navigation
- Lightweight and fast

## 📦 Requirements

- WordPress
- WooCommerce
- PW WooCommerce Gift Cards plugin

## 🔧 Installation

1. Download or clone this repository
2. Upload the plugin folder to `/wp-content/plugins/`
3. Activate the plugin via the WordPress dashboard
4. Make sure WooCommerce and PW Gift Cards plugin are active

## 🧩 How It Works

This plugin registers a custom endpoint in the WooCommerce **My Account** page and retrieves gift card data associated with the current user. The data is then displayed in a structured and user-friendly format.

## ⚠️ Notes

- This plugin extends the functionality of the PW Gift Cards plugin.
- It does not modify any core plugin files.
- Compatibility depends on the data structure provided by the PW Gift Cards plugin.
